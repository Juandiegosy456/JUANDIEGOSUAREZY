Propósito

Definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores,sirven para indicar qué son y cuál es el significado de los elementos que contienen dentro composición de un documento web

Etiquetas semánticas

Son utilizadas para mejorar la creación de un documento web, que sirven para definir la estructura de un sitio y cómo nos pueden ayudar a que nuestra página sea indexada con mayor exactitud por los motores de búsqueda.

Listas

Permite crear lista despegable de trabajo a realizar, permitiendo insertar todo lo necesario para tener la informacion detallada, con el fin de manejar un orden a la pagina web, separando informacion con otra. (nombreLista)

Enlaces

Primordialmente sobre los enlaces de html que nos permite dar una conexion de un lugar con otro, puede ser pagina web, imagenes, videos, URL genérica o archivo. Con el fin de enlazar todos los elementos necesario para funcionar una pagina web.
